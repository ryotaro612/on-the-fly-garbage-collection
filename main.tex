\section{On-the-Fly Garbage Collection}
\begin{frame}
  \frametitle{有向グラフによるヒープの表現}
  {\large ルートから到達できないノードをガベージとみなす}
  \begin{itemize}
  \item 大域やスタックの変数が参照する値はルート
  \item ガベージノードを白に着色
  \item ルートから到達できる中間ノードや葉を黒に着色
  \item ノード数は固定
  \end{itemize}
\end{frame}
